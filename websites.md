Search extensively on the basis of these links below, for each archive separately and potentially combine standards if archives have common API endpoints, if you can find easy API endpoints that download these documents. If not, and for the links below for which you cannot, use the playwright mcp to write a script that downloads all of these. You can also use the playwright mcp to explore these websites and links if these lead you to clear api endpoints.

We want to download ONLY the memories van successie. Not a separate appendix called the Tafel V-bis.
